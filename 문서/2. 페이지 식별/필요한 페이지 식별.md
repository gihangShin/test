# 필요한 페이지 식별 정리

## 공통
1. 메인페이지
2. 게시판 페이지 (복붙)
  - 공지사항
  - 운영자 choice
  - 자유게시판
  - 추가 가능 게시판
3. 게시글 페이지 (read, delete)
  - 댓글 (게시글 페이지 내 modal)
4. 글 작성 페이지 (insert)
5. 글 수정 페이지 (update, delete)
6. 검색 결과 페이지

- 공통코드 보류()
